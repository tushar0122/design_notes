Application Layer Protocol 
    => These protocols enable applications to communicate over a network.

HTTP/HTTPS (Hypertext Transfer Protocol Secure)
    => Web browsing.
FTP (File Transfer Protocol)
    => File sharing.
SMTP (Simple Mail Transfer Protocol)
    => Email sending.
IMAP/POP3 (Internet Message Access Protocol/Post Office Protocol)
    => Email retrieval.
MQTT (Message Queuing Telemetry Transport)
    => IoT communication

WebSocket (WS/WSS)	
    => Full-duplex, real-time communication over a single TCP connection	
Server Side Events 
    => Push real-time updates from server to client
